# Grinch Invasion

## A cozy Space Invaders-like pygame with a Christmas theme!
Save Christmas from Grinch invasion: as a snowman, you need to stop them throwing Christmas sugar canes. Test you speed and ability and set your score record!
This project, whose purpose is just learning and having fun, is based on the "Alien Invasion" project on the Python Crash Course volume by Eric Matthes. 

## Controls
- Click the button with your mouse to start the game
- Use your keyboard arrow keys to move the snowman
- Space bar is for shooting the sugar canes
- Press Enter to pause the game
- Press Esc to exit the game
